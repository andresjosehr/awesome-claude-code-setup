# Confirmar y Subir Trabajo

Por favor realiza las siguientes tareas:

1. **Verificar estado de git actual** - Revisar todos los archivos modificados y entender que cambios se han realizado

2. **Crear commit significativo** - Agregar los archivos apropiados al staging y crear un commit con un mensaje claro y descriptivo siguiendo las convenciones del proyecto

3. **Subir al remoto** - Hacer push de tus commits a la rama apropiada

4. **Verificar feedback de PR** - Revisar cualquier pull request abierto por:
   - Comentarios de revisores
   - Solicitudes de cambios
   - Estado del pipeline CI/CD
   - Conflictos de merge

5. **Planificar acciones de seguimiento** - Basado en cualquier feedback encontrado:
   - Crear una lista de tareas para atender los comentarios de revision
   - Identificar cualquier problema bloqueante
   - Anotar cualquier pregunta para el equipo

Recuerda:
- Seguir el formato de mensaje de commit usado en este repositorio
- Agrupar cambios relacionados en commits logicos
- Asegurar que las pruebas pasen antes de hacer push
- Usar PRs en borrador para trabajo en progreso
